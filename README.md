# Windows Service Health Checker

A read-only PowerShell toolkit for Windows service status review.

## Features

- Key Windows service status summary
- Startup type context
- Dependency information
- Service inventory exports
- CSV, JSON, and HTML reports

## How to run

```powershell
powershell.exe -ExecutionPolicy Bypass -File .\Windows_Service_Health_Checker.ps1
```

## Safety

Diagnostic-only. It reports service information for support review.
